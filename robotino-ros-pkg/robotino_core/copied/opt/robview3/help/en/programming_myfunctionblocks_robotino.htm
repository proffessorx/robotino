Use on Robotino

       

It is also possible to use your own function blocks on Robotino. To do this it is necessary to build the plugins for the Linux system installed on Robotino. The new 4GB flash card for Robotino contains all necessary Software (C++ compiler, make, CMake, RobotinoView API), so the plugins can be built directly on Robotino. For bigger projects it is recommended to use a development system with a faster CPU and more memory; it is recommended to use Ubuntu 9.04 as operating system for developping.

The own function blocks are located in the directory /home/robotino/Festo/RobotinoView3/units/MyFunctionBlocks.

To build a plugin please follow these steps:

•Copy the folder with the source files on Robotino  into the folder "/home/robotino/Festo/RobotinoView3/MyFunctionBlocks" via FTP (user: robotino, password: robotino). The path "/home/robotino/Festo/RobotinoView3/MyFunctionBlocks" is created the first tome Robotino View or the Interpreter is launched.
•Connect to Robotino via SSH (user: robotino, password: robotino).
•Create a working directory (for Tutorial 1, e.g.) and cd to it:
 
mkdir /home/robotino/build/tutorial1
cd /home/robotino/build/tutorial1
•Create the makefile from the CMake scripts:
 
cmake /home/robotino/Festo/RobotinoView3/MyFunctionBlocks/tutorial1.unit/plugin
 
Alternatively it, similarly to Windows, possible to specify other settings like compiler flags, e.g., by using
 
ccmake /home/robotino/Festo/RobotinoView{MajorVer}/MyFunctionBlocks/tutorial1.unit/plugin.
 
To do this, the project must first be configured (key "C"). Than the settings can be adapted. After another keystroke on "C" and one on "G" ("Generate") the Makefile will be created automatically.
•Compile and link the simulation and gui plugin:
 
make
 
The libraries will be copied to the right place automatically after building them.
•Add the function block to the description file of the "My function blocks" device. To do that, the file /home/robotino/Festo/RobotinoView3/devices/MyFunctionBlocks/device.xml must be edited. Alternatively the file can be edited on Windows using the "Function blick manager" and copied on Robotino via FTP.
