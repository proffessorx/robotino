Creating a new function block

       

To create a new function block the tool "Function blick manager" is included in the Robotino® View API. It can be launches via the Windows start menu (in the RobotinoView folder). To create a function block, change to the tab "Function block designer".

In the example the function block "Test function block" is created. It has an input and an output, both of type uint32, no state variables and a dialog. By the way, it is not necessaty to enter all data. It is sufficient wo specify "Vendor" and "Name". After clicking on "Auto fill" the other fields (except "Icon file name") will be filled automatically with appropriate data. Then it is only necessary to specify the icon, inputs, outputs, state variables and if the function blick is supposed to have a dialog.

After clicking on "Generate" the new function block is created. In the folder %USERPROFILE%\Festo\RobotinoView3\units\MyFunctionBlocks a new subdirectory is created that contains the XML description file, CMake scripts and even C++ source files (without any function). So it is possible to create a Visual C++ project with CMake and work with it.

 

For the function blick to be visible in Robotino View it must be added to the device "My function blicks". To do this, change to the tab "Device manager".

Subfolders and function blocks can be added and removed by using the appropriate functions in the context menu; they can be moved by Drag & Drop. Subfolders can be renamed directly in the view. After finishing all modifications, the new state is saved by clicking the "Save" button.
