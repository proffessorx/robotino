Grappler for Robotino®

       

The picture shows the Grappler for Robotino®. The grappler is made of four axes  (A1 to A4). With the push buttons B1 and B2 the user is able to perform input operations directly at the grappler.

Push button

Function

B1

Store the current configuration.

B2

Toggle normal operation and free floating oepration.

The positions selector funktion block is used to drive to previously stored positions (see Manipulator dialog). The positions selector uses the position number or name as input.

Position selector with position name
