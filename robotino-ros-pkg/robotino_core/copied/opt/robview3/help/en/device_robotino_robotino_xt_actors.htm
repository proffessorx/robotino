Actors

       

The compact version of the bionic handling assistant is driven by eight bellows plus gripper. Robotino View allows to set the pressure in each bellows. The gripper is activated by the function blocks of the gripper valves. To simplify the movement of the arm three function blocks are provided:

•Segment XY: Calculates pressures from xy-coordinates.
•Gripper rotate: Translates a single input into pressure for the corresponding two bellows.
•Gripper open: Translates a single input into state of the two relays.
