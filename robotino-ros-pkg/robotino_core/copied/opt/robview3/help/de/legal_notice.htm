Rechtliche Hinweise

Diese Arbeit basiert auf oder enthält Arbeit Dritter oder liefert solche Werke mit aus. Diese Werke sind im Folgenden aufgelistet. Die Bereitstellung der Werke unterliegt möglicherweise eigenen Bedingungen, die in der Robotino® View-Lizenzvereinbarung genannt werden, die bei der Installation von Robotino® View angezeigt wird.

•Boost C++ Libraries: Diese Arbeit basiert teilweise auf Boost, einem Paket von freien, portablen C++-Bibliotheken. Unter der Adresse http://www.boost.org/ finden Sie weitere Informationen über Boost.
•CMake: Diese Arbeit beinhaltet eine binäre Distribution von CMake, einem Open-Source-System, das Build-Prozesse in betriebssystem- und Compiler-unabhängiger Weise verwaltet. Unter der Adresse http://cmake.org/ finden Sie weitere Informationen über CMake.
•Xerces-C++ XML Parser: Diese Arbeit basiert teilweise auf dem Xerces-C++ XML Parser. Unter der Adresse http://xerces.apache.org/xerces-c/ finden Sie weitere Informationen über Xerces-C++.
•Lua und LuaJIT: Diese Arbeit basiert teilweise auf dem Lua-Interpreter bzw. dem Lua JIT-Compiler. Unter den Adressen http://www.lua.org und http://www.luajit.org finden Sie weitere Informationen über Lua und LuaJIT.

Bitte entnehmen Sie die exakten Lizenzbedingungen für Werke Dritter der Robotino® View-Lizenzvereinbarung, die bei der Installation von Robotino® View angezeigt wird.
