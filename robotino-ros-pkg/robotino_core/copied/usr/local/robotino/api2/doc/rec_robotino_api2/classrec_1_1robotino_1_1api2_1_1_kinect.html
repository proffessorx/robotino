void rec::robotino::api2::Kinect::setKinectNumber ( int  number)

Sets the number of this Kinect device.

Set the number of this Kinect to number. The default Kinect number is 0. Setting the Kinect number only makes sense when your Robotino is equipped with more than one Kinect.

Parameters
numberThe Kinect number. Range [0; numKinects()-1]. When setting number < 0 no Kinect is selected.
Exceptions
RobotinoExceptionif the given Kinect number is out of range. RobotinoException if given com object is invalid.
