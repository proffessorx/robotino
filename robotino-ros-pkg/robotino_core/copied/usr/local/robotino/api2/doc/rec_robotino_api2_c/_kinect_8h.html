Detailed Description

In "rec/robotino/api2/c/Kinect.h" you can find functions for reading Robotino's laser rangefinder.

Use Kinect_construct() to create a new rangefinder object. Associate the rangefinder object with a com object using Kinect_setComId(). Use Kinect_getReadings() to get sensor readings from Robotino's rangefinder.

Get readings from Robotino's kinect. Do not forget to call Kinect_grab first. Get the data sizes by calling Kinect_dataSize first. The following conditions must be met:

  • sizeof(depth_data) >= depthDataWidth * depthDataHeight * sizeof(float)
  • sizeof(object_data) >= objectDataWidth * objectDataHeight * sizeof(float) The easiest way to meet these conditions is to create arrays containing more elements than a rangefinder will provide readings: float depth_data[320*240]; float object_data[320*240];
    Parameters
    idThe kinect id.
    depth_dataDepth information. See parameter format.
    object_dataRobotinoSIM provides object IDs here. This can be used to easily find depth readings belonging to the same object.
    depthDataWidthdata width.
    depthDataHeightdata height.
    objectDataWidthobject data width.
    objectDataHeightobject data height.
    formatFormat: 0-standard 11bit kinect format, 4-16bit mm values
    stampTimestamp
    Returns
    Returns TRUE (1) on success. Returns FALSE (0) if the given KinectId is invalid.
